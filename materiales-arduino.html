Placas Arduino

Arduino UNO

La placa más popular y versátil. Cuenta con microcontrolador ATmega328P, perfecta para principiantes y proyectos educativos.

Especificaciones Técnicas
  • Microcontrolador: ATmega328P
  • 14 pines digitales I/O
  • 6 entradas analógicas
  • Voltaje de operación: 5V
  • Memoria Flash: 32KB

Arduino MEGA

Versión más grande con ATmega2560, ideal para proyectos complejos que requieren más entradas/salidas y memoria adicional.

Especificaciones Técnicas
  • Microcontrolador: ATmega2560
  • 54 pines digitales I/O
  • 16 entradas analógicas
  • Memoria Flash: 256KB
  • 4 puertos UART

Arduino NANO

Versión compacta con las mismas capacidades del UNO pero en formato reducido. Perfecta para proyectos donde el espacio es limitado.

Especificaciones Técnicas
  • Microcontrolador: ATmega328P
  • Tamaño compacto: 18x45mm
  • Conexión: Mini-USB
  • Compatible con breadboard
  • Voltaje: 5V

Componentes Básicos

LEDs

Diodos emisores de luz en varios colores. Componentes fundamentales para indicadores visuales y proyectos de iluminación.

Características
  • Voltaje: 2-3.3V según color
  • Corriente: 20mA máximo
  • Colores: rojo, verde, azul, RGB
  • Requiere resistencia limitadora

Resistencias

Limitan el flujo de corriente en el circuito. Esenciales para proteger componentes. Se identifican por bandas de colores.

Valores Comunes
  • 220Ω - 330Ω para LEDs
  • 1kΩ - 10kΩ para pull-up/down
  • Potencia: 1/4W o 1/2W
  • Tolerancia: ±5%

Pulsadores

Interruptores momentáneos que permiten la entrada de usuario. Se usan con resistencias pull-up o pull-down.

Tipos
  • Tactile switch (común)
  • Push button con retención
  • Conexión: normalmente abierto
  • Vida útil: 1 millón de ciclos

Potenciómetros

Resistencias variables que permiten leer valores analógicos. Perfectos para controles de velocidad, volumen o ajustes.

Especificaciones
  • Valores: 1kΩ, 10kΩ, 100kΩ
  • 3 terminales: VCC, GND, señal
  • Rotación: 270° típico
  • Lectura: 0-1023 en Arduino

Cables Jumper

Cables de conexión con diferentes terminaciones. Esenciales para conectar componentes en el protoboard.

Tipos
  • Macho-Macho (breadboard)
  • Macho-Hembra (Arduino-sensor)
  • Hembra-Hembra (módulos)
  • Longitud: 10-30cm

Protoboard

Placa de pruebas sin soldadura. Permite crear circuitos temporales conectando filas y columnas.

Características
  • Puntos de contacto: 400-830
  • Líneas de alimentación
  • Canal central para chips
  • Sin soldadura requerida

Sensores

Sensor de Temperatura

DHT11 o DHT22 para medir temperatura y humedad. Esencial para proyectos de domótica y monitoreo ambiental.

Especificaciones
  • Rango temperatura: -40°C a 80°C
  • Humedad: 0-100% RH
  • Precisión: ±0.5°C (DHT22)
  • Comunicación: 1-Wire digital

Sensor Ultrasónico HC-SR04

Mide distancias mediante ondas ultrasónicas. Ideal para robots, sistemas anti-colisión y detección de objetos.

Especificaciones
  • Rango: 2cm a 4 metros
  • Precisión: 3mm
  • Ángulo: 15 grados
  • Pines: VCC, Trig, Echo, GND

Fotoresistencia LDR

Sensor de luz que varía su resistencia según la iluminación. Usado en sistemas de iluminación automática.

Características
  • Resistencia: 1MΩ (oscuridad)
  • Resistencia: 1-10kΩ (luz)
  • Tiempo de respuesta: 8-12ms
  • Requiere divisor de voltaje

Sensor PIR

Detector de movimiento infrarrojo pasivo. Detecta cambios de calor por movimiento humano.

Especificaciones
  • Rango: 3-7 metros ajustable
  • Ángulo: 120 grados
  • Delay ajustable: 0.3s-5min
  • Salida: Digital HIGH/LOW

Actuadores

Servomotor SG90

Motor con control de posición preciso. Gira 180° mediante señales PWM. Ideal para robótica y mecanismos.

Especificaciones
  • Voltaje: 4.8V - 6V
  • Torque: 1.8 kg/cm a 4.8V
  • Velocidad: 0.12 seg/60°
  • Ángulo: 0° a 180°

Motor DC

Motor de corriente continua para rotación continua. Requiere driver L298N para control desde Arduino.

Especificaciones
  • Voltaje: 3V - 12V
  • Velocidad: 3000-6000 RPM
  • Control: PWM para velocidad
  • Requiere driver de potencia

Buzzer Piezoeléctrico

Zumbador para emitir sonidos. El pasivo permite control de frecuencia para tonos musicales y melodías.

Tipos
  • Activo: tono fijo al energizar
  • Pasivo: control de frecuencia
  • Voltaje: 3.5V - 5V
  • Rango: 2kHz - 4kHz

Módulo Relé

Interruptor electromagnético para controlar circuitos de alta potencia (110V/220V) con señal de 5V.

Especificaciones
  • Control: 5V DC
  • Carga máx: 10A a 250V AC
  • Tipos: 1, 2, 4, 8 canales
  • LED indicador incluido

Comunicación y Displays

Módulo Bluetooth HC-05

Comunicación inalámbrica con smartphones y computadoras. Usa comunicación serial UART.

Especificaciones
  • Versión: Bluetooth 2.0
  • Alcance: hasta 10 metros
  • Velocidad: 9600 bps default
  • Voltaje: 3.3V (incluye regulador)

Display LCD 16x2

Pantalla de cristal líquido de 2 líneas y 16 caracteres. Con módulo I2C solo requiere 2 pines.

Características
  • Caracteres: 16 columnas x 2 filas
  • Con backlight ajustable
  • Módulo I2C opcional
  • Voltaje: 5V

Módulo WiFi ESP8266

Conectividad WiFi para proyectos IoT. Puede funcionar standalone o como shield de Arduino.

Especificaciones
  • WiFi: 802.11 b/g/n
  • Procesador: 80MHz
  • Memoria: 4MB Flash
  • Pines GPIO: 17 disponibles

Joystick Analógico

Control de dos ejes (X, Y) con botón integrado. Devuelve valores analógicos 0-1023 en cada eje.

Especificaciones
  • Ejes: X e Y analógicos
  • Botón: pulsador digital
  • Valores: 0-1023 por eje
  • Voltaje: 5V

Componentes Adicionales

Transistores

NPN (2N2222, BC547) para amplificar señales y controlar componentes de alto consumo. Actúan como interruptores electrónicos.

Modelos Comunes
  • 2N2222: NPN, 600mA, 40V
  • BC547: NPN, 100mA, 45V
  • TIP120: NPN Darlington, 5A
  • Para control de motores y LEDs

Condensadores

Almacenan energía temporalmente. Electrolíticos para fuente de poder, cerámicos para filtrado de ruido.

Tipos
  • Electrolíticos: 10µF-1000µF
  • Cerámicos: 100nF para filtrado
  • Polarizados: respetar + y -
  • Voltaje: según aplicación

Fuente de Alimentación

Adaptador de 9V-12V DC para alimentar Arduino externamente. Necesario para proyectos con alto consumo.

Especificaciones
  • Voltaje: 9V-12V DC
  • Corriente: 1A-2A típico
  • Conector: Jack 2.1mm
  • Polaridad: centro positivo

Multímetro Digital

Herramienta esencial para medir voltaje, corriente y resistencia. Indispensable para diagnosticar problemas.

Funciones
  • Voltaje AC/DC
  • Corriente AC/DC
  • Resistencia (Ω)
  • Continuidad y diodos
