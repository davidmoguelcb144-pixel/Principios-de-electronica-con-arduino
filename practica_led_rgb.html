¿Por Qué Funciona?

🔴

Teoría de Color RGB

El LED RGB contiene tres diodos emisores de luz: Rojo (Red), Verde (Green) y Azul (Blue). Al combinar estos tres colores primarios en diferentes intensidades, podemos crear millones de colores diferentes.

⚡

Control PWM

El microcontrolador A6 utiliza modulación por ancho de pulso (PWM) para controlar la intensidad de cada color. Variando el ciclo de trabajo entre 0% y 100%, regulamos el brillo de cada LED interno.

🔌

Resistencias Limitadoras

Las resistencias (código de colores: naranja-naranja-café) protegen cada LED limitando la corriente. Sin ellas, el exceso de corriente destruiría los diodos. Calculamos su valor usando la Ley de Ohm.

🔄

Cátodo Común

Este LED RGB tiene configuración de cátodo común (pin central a tierra). Los pines externos se conectan a las salidas digitales del microcontrolador a través de las resistencias, permitiendo controlar cada color independientemente.

💾

Programación Digital

El código cargado en el A6 define secuencias de colores mediante valores digitales (0-255) para cada canal RGB. Esta señal digital se convierte en una señal analógica mediante PWM.

🌈

Mezcla Aditiva

A diferencia de la mezcla de pinturas, la luz sigue el modelo aditivo: Rojo + Verde = Amarillo, Verde + Azul = Cian, Rojo + Azul = Magenta, y los tres juntos crean blanco.

Componentes Utilizados

🎛️

Microcontrolador A6

Cerebro del circuito, genera señales PWM para controlar el LED RGB

💡

LED RGB Cátodo Común

Componente emisor de luz con tres colores integrados (Rojo, Verde, Azul)

📊

Resistencias 330Ω

Tres resistencias (naranja-naranja-café) para limitar corriente en cada canal

🔲

Protoboard

Placa de pruebas para ensamblar el circuito sin soldadura

🔗

Cables Jumper

Cables de conexión: rojos (positivo), negro (tierra), azul y amarillo (señales)

Especificaciones Técnicas
