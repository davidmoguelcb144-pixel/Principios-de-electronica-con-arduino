¿Qué es un Servomotor?

Un servomotor es un actuador rotativo que permite un control preciso de posición angular, velocidad y aceleración. A diferencia de los motores DC convencionales, los servomotores incorporan un sistema de retroalimentación que les permite mantener su posición con gran exactitud.

Los servomotores se utilizan ampliamente en robótica, automatización, sistemas de control y proyectos de electrónica por su capacidad de moverse a posiciones específicas con precisión y mantenerlas mediante un circuito de control interno.

Materiales Utilizados

Para realizar esta práctica de control de servomotor con Arduino, se requieren los siguientes componentes electrónicos:

🔵
Arduino UNO
Microcontrolador que genera las señales PWM necesarias para controlar el servomotor. Proporciona alimentación de 5V y gestiona la comunicación.
⚙️
Servomotor SG90
Servomotor de 180° con torque de 1.8 kg/cm. Incluye tres cables: rojo (alimentación), café/negro (tierra), naranja/amarillo (señal de control).
🔌
Cable USB
Cable USB tipo A a tipo B para conectar el Arduino al computador. Permite la programación y suministra alimentación de 5V.
🔗
Cables Jumper
Cables de conexión macho-hembra para conectar el servomotor directamente a los pines del Arduino. Tres cables de colores según función.

Diagrama de Conexión

Esquema Eléctrico Arduino + Servomotor

Conexiones: Cable naranja → Pin digital PWM (~), Cable rojo → 5V, Cable café → GND

Proceso de Conexión Paso a Paso

A continuación se muestra el proceso detallado de conexión del servomotor al Arduino, identificando cada cable por su función específica.

1

Cable Café - Conexión a Tierra (GND)

El cable café (o negro en algunos modelos) es la conexión a tierra (GND) del servomotor. Este cable debe conectarse a cualquier pin GND del Arduino para cerrar el circuito eléctrico. La tierra es el punto de referencia común para todas las señales eléctricas del sistema.

2

Cable Rojo - Alimentación +5V

El cable rojo corresponde a la alimentación positiva del servomotor. Debe conectarse al pin de 5V del Arduino. Este voltaje alimenta tanto el circuito de control interno del servo como el motor DC que genera el movimiento. La corriente máxima del SG90 es de aproximadamente 500mA en movimiento.

3

Cable Naranja - Señal PWM de Control

El cable naranja (o amarillo/blanco) es la línea de señal de control. Debe conectarse a un pin digital PWM del Arduino (identificados con el símbolo ~). Por este cable, el Arduino envía pulsos de ancho modulado (PWM) que indican al servomotor la posición angular deseada. La señal típicamente tiene una frecuencia de 50Hz con pulsos de 1-2ms de duración.

¿Por Qué Funciona el Servomotor?

El funcionamiento del servomotor se basa en la combinación de varios principios de control, electrónica y mecánica. A continuación se explican los conceptos fundamentales:

⚡ Principios de Funcionamiento

  • Control por Modulación de Ancho de Pulso (PWM): El Arduino genera pulsos eléctricos de duración variable (típicamente entre 1ms y 2ms) a una frecuencia de 50Hz (20ms de período). La duración del pulso determina la posición angular del servo: 1ms = 0°, 1.5ms = 90°, 2ms = 180°.
  • Circuito de Control Interno: El servomotor contiene un circuito electrónico que decodifica la señal PWM recibida y la compara con la posición actual detectada por el potenciómetro interno. Este circuito genera las señales necesarias para controlar el motor DC.
  • Motor DC y Sistema de Engranajes: Dentro del servomotor hay un pequeño motor de corriente continua acoplado a una caja reductora de engranajes. Los engranajes reducen la velocidad y aumentan el torque, permitiendo movimientos precisos y con fuerza suficiente para aplicaciones prácticas.
  • Retroalimentación por Potenciómetro: Un potenciómetro acoplado al eje de salida convierte la posición mecánica en una señal eléctrica proporcional. El circuito de control compara esta señal con la posición deseada (indicada por el PWM) y ajusta el motor hasta alcanzar la posición objetivo.
  • Sistema de Control en Lazo Cerrado: La combinación de sensor (potenciómetro), controlador (circuito interno) y actuador (motor DC) forma un sistema de control en lazo cerrado. Este sistema corrige automáticamente cualquier error de posición causado por cargas externas o perturbaciones.
  • Fuente de Alimentación: El pin de 5V del Arduino suministra la energía necesaria tanto para el circuito de control como para el motor. La corriente puede variar entre 100mA (en reposo) hasta 500-800mA (bajo carga máxima). Para aplicaciones con múltiples servos o cargas pesadas, se recomienda usar una fuente de alimentación externa.
  • Codigo usado: // C++ code //usamos librería para servo #include Servo grados; //declaración de variables int pinservo=9; //pin para comunicar el servo //definimos las posiciones int pos1=0;//0 grados int pos2=90;//90 grados int pos3=180;//180 grados int posicion=0;//posicion inicial void setup() { //conectando el servo grados.attach(pinservo);//indicamos en que pin se conecta } void loop() { posicion=pos1;//0 grados grados.write(posicion);//ajustamos la posicion indicada delay(2000); //pausa antes del siguiente movimiento posicion=pos2; grados.write(posicion); delay(2000); posicion=pos3; grados.write(posicion); delay(2000); }

🔬 Especificaciones Técnicas SG90

  • Voltaje de operación: 4.8V - 6V (típicamente 5V)
  • Corriente sin carga: 100mA @ 5V
  • Corriente con carga: 500-800mA @ 5V
  • Torque máximo: 1.8 kg/cm @ 5V
  • Velocidad: 0.1s/60° @ 5V
  • Rango de movimiento: 0° - 180° (aproximadamente)
  • Frecuencia de señal: 50Hz (período de 20ms)
  • Ancho de pulso: 1ms (0°) a 2ms (180°)
  • Peso: 9 gramos
  • Dimensiones: 22 x 11.5 x 27 mm
