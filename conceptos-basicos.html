¿Qué es Arduino?

Definición

Arduino es una plataforma de electrónica de código abierto basada en hardware y software flexibles y fáciles de usar. Está pensada para que artistas, diseñadores, aficionados y cualquier persona interesada en crear objetos y entornos interactivos pueda hacerlo sin necesidad de conocimientos avanzados en electrónica.

La plataforma combina una placa microcontroladora (el hardware físico) con el entorno de desarrollo Arduino IDE (el software) en el que se escribe el código. Los programas escritos para Arduino se llaman sketches.

El microcontrolador más usado históricamente es el ATmega328P (en el Arduino Uno), aunque existen modelos con otros chips. Arduino puede leer entradas —sensores, pulsadores, señales— y convertirlas en salidas: encender un LED, activar un motor, publicar datos en internet, y mucho más.

Componentes de Hardware
🔲

Microcontrolador

Chip central que ejecuta el programa. En el Arduino Uno es el ATmega328P a 16 MHz con 32 KB de Flash, 2 KB de SRAM y 1 KB de EEPROM.

⚡

Alimentación

Puede alimentarse por USB (5 V) o por el conector de barril (7–12 V recomendado). Incluye reguladores internos para proveer 5 V y 3.3 V a los periféricos.

📌

Pines Digitales

14 pines digitales (0–13) que operan a 5 V. Pueden configurarse como entrada (INPUT) o salida (OUTPUT). Algunos tienen capacidad PWM (~).

〰️

Pines Analógicos

6 entradas analógicas (A0–A5) con un convertidor ADC de 10 bits (0–1023). Permiten leer señales continuas de sensores como potenciómetros o fotorresistencias.

🔌

Puerto USB / Serie

Usado para cargar sketches y comunicarse con la PC mediante el Serial Monitor. En el Uno, el chip ATmega16U2 actúa como convertidor USB-Serie.

💡

LED integrado

El pin 13 está conectado a un LED integrado en la placa, muy útil para pruebas rápidas sin necesitar componentes externos. Es el clásico "Blink".

🔁

Botón de Reset

Reinicia el microcontrolador y vuelve a ejecutar el sketch desde el principio, equivalente a desconectar y reconectar la alimentación.

🧩

Conectores / Shields

Los shields son placas de expansión apilables que añaden funcionalidades: WiFi, Ethernet, motor driver, LCD, GPS, etc., sin necesidad de soldar.

🔶

Cristal Oscilador

Cristal de cuarzo de 16 MHz que proporciona la señal de reloj al microcontrolador, determinando la velocidad de ejecución de instrucciones.

Arduino IDE
🖥️

Entorno de Desarrollo

El Arduino IDE es el editor oficial gratuito donde se escribe, compila y carga el código a la placa. Disponible en versión de escritorio y web (Arduino Cloud Editor).

🔍

Serial Monitor

Herramienta integrada que muestra en tiempo real los datos enviados con Serial.print(). Imprescindible para depurar y monitorear sensores.

📦

Gestor de Librerías

Desde Sketch → Incluir Librería → Gestionar Librerías se pueden instalar librerías de terceros para sensores, pantallas, motores y protocolos de comunicación.

🗂️

Gestor de Placas

Permite añadir soporte para placas de terceros como ESP32, ESP8266, STM32 o Raspberry Pi Pico instalando los board packages correspondientes.

Estructura de un Sketch

Las dos funciones obligatorias

Todo programa de Arduino tiene al menos dos funciones: setup() se ejecuta una sola vez al arrancar, y loop() se repite indefinidamente.

SKETCH EJEMPLO 
// ── Declaraciones globales ──────────────────────────────
const int LED  = 13;    // pin del LED integrado
int       vel  = 0;     // velocidad de parpadeo (ms)

// ── setup(): se ejecuta UNA sola vez ────────────────────
void setup() {
  pinMode(LED, OUTPUT);   // configura LED como salida
  Serial.begin(9600);      // inicia comunicación serie
  Serial.println("Arduino listo!");
}

// ── loop(): se repite CONTINUAMENTE ─────────────────────
void loop() {
  vel = analogRead(A0) / 4; // lee potenciómetro (0–255)

  digitalWrite(LED, HIGH);  // enciende LED
  delay(vel);                 // espera
  digitalWrite(LED, LOW);   // apaga LED
  delay(vel);                 // espera

  Serial.print("Velocidad: ");
  Serial.println(vel);
}
Funciones Principales
📥

pinMode()

Configura un pin como INPUT, OUTPUT o INPUT_PULLUP. Se llama generalmente en setup().

📤

digitalWrite()

Escribe HIGH (5 V) o LOW (0 V) en un pin digital configurado como salida.

🔎

digitalRead()

Lee el estado de un pin digital: devuelve HIGH o LOW. Útil para botones e interruptores.

〰️

analogRead()

Lee un pin analógico (A0–A5) y devuelve un valor entre 0 y 1023 (10 bits), proporcional al voltaje 0–5 V.

📉

analogWrite()

Genera una señal PWM en pines compatibles (~). El valor va de 0 (0%) a 255 (100% ciclo de trabajo).

⏱️

delay() / millis()

delay(ms) pausa la ejecución. millis() devuelve el tiempo transcurrido desde el arranque sin bloquear el programa.

🗺️

map()

Reescala un valor de un rango a otro: map(val, 0, 1023, 0, 255). Muy usado para convertir lecturas analógicas a otros rangos.

📡

Serial

Serial.begin(), Serial.print() y Serial.read() gestionan la comunicación serie con la PC u otros dispositivos.

🎚️

constrain() y abs()

constrain(x, min, max) limita un valor dentro de un rango. abs(x) devuelve el valor absoluto de un número.

Librerías Populares

Sensores y Actuadores

  • DHT.h — Temperatura y humedad con DHT11/DHT22
  • Servo.h — Control de servomotores con posición exacta
  • Stepper.h — Control de motores paso a paso
  • NewPing.h — Sensor ultrasónico HC-SR04
  • IRremote.h — Receptor y emisor infrarrojo
  • FastLED / NeoPixel — Tiras LED RGB WS2812B

Comunicación y Pantallas

  • Wire.h — Protocolo I²C nativo
  • SPI.h — Protocolo SPI nativo
  • LiquidCrystal.h — Pantallas LCD 16×2 en paralelo
  • LiquidCrystal_I2C — LCD con módulo I²C
  • Adafruit GFX — Gráficos en pantallas TFT/OLED
  • SoftwareSerial — Puerto serie en cualquier pin
Modelos Más Usados
🟦

Arduino Uno

El modelo de referencia para aprender. ATmega328P, 14 pines digitales, 6 analógicos. Ideal para proyectos educativos y prototipado.

🟩

Arduino Nano

Mismo chip que el Uno pero en formato compacto. Perfecto para proyectos donde el espacio es limitado; se conecta directamente a una protoboard.

🔵

Arduino Mega 2560

Para proyectos grandes: 54 pines digitales, 16 analógicos, 4 puertos serie. ATmega2560 con 256 KB de Flash. Compatible con más shields.

📶

Arduino MKR / Nano 33

Placas modernas con conectividad WiFi, BLE o LoRa integrados. Ideal para proyectos IoT. Operan a 3.3 V (no son directamente 5 V tolerantes).

⚙️

ESP32 / ESP8266

Compatibles con el ecosistema Arduino mediante el gestor de placas. Incluyen WiFi y Bluetooth integrados a un precio muy bajo. Muy populares en IoT.

🎛️

Arduino Due

Basado en ARM Cortex-M3 a 84 MHz, opera a 3.3 V y ofrece DAC real (salida analógica auténtica), perfecto para audio y procesamiento de señales.
